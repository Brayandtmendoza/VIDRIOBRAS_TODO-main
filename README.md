# VIDRIOBRAS

Sistema de gestión para VIDRIOBRAS.

## Estructura del Proyecto

- `backend/` - Aplicación Flask (Python)
- `frontend/` - Aplicación React

## Instalación

### Backend
```bash
cd backend
pip install -r requirements.txt
```

### Frontend
```bash
cd frontend/Reac/mi-proyecto
npm install
```
